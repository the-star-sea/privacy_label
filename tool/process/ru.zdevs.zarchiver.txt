=== Data Collection and Use
ZArchiver does not collect, transfer, sell personal information such as your name, email address, phone number, credit card, information about usage app or any other data. ZArchiver store archive passwords for some time to access encrypted files. Saved password are stored encrypted, but a hack-proof confidentiality is not guaranteed. === Permissions with explanation
READ_EXTERNAL_STORAGE and WRITE_EXTERNAL_STORAGE (Android pre 11) or MANAGE_EXTERNAL_STORAGE (Android 11+) - to read and write files on external storage (SD card, USB flash);
WAKE_LOCK - to proper work of long-lasting operations;
INSTALL_PACKAGES - to install APK files in Android 8+;
FOREGROUND_SERVICE - to run of file operations;
POST_NOTIFICATIONS - to show progress notification of a background process. === Changes to This Privacy Policy
